# @sqirrel-forge/ui-util
A collection of utilities, classes, functions and abstracts made for the browser and babel compatible.

## Installation

```
npm i @squirrel-forge/ui-util
```

## Usage

```
import { Class, function } from '@squirrel-forge/ui-util';
```
